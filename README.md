**Étapes du projet**

- Modélisation : Construire le modèle.
- Analyse théorique : Étudier la probabilité stationnaire pour 1 vélo et 5 stations (voir section 2.3 du livre de Kelly et Yudovina).
- Simulation : Écrire le programme et vérifier la correspondance entre théorie et simulation.
- Exécution : Faire tourner le programme pour 100 vélos et 5 stations (paramètres fournis).
